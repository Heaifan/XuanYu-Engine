# UI 术语表 — 9.1B

用于统一中文显示、英文标识、代码命名。

## 规则

- 代码命名：英文
- UI 默认显示：中文
- 文档术语：中英文并列
- 完整多语言切换：后续再做

## 菜单

| 中文显示 | English | Code Key | 用途 |
|---|---|---|---|
| 文件 | File | Menu_File | 新建、打开、保存、退出 |
| 编辑 | Edit | Menu_Edit | 撤销、重做、复制、粘贴、删除 |
| 视图 | View | Menu_View | 显示/隐藏面板、网格、辅助线 |
| 窗口 | Window | Menu_Window | 面板布局、重置窗口 |
| 帮助 | Help | Menu_Help | 文档、快捷键、关于 |

## 命令

| 中文显示 | English | Code Key | 用途 |
|---|---|---|---|
| 撤销 | Undo | Command_Undo | 撤销上一步操作 |
| 重做 | Redo | Command_Redo | 重做已撤销的操作 |
| 保存 | Save | Command_Save | 保存当前工程/地图 |
| 重置布局 | Reset Layout | Command_ResetLayout | 恢复默认布局 |
| 运行 | Play | Command_Play | 运行模拟 |
| 停止 | Stop | Command_Stop | 停止模拟 |
